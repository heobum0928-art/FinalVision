
**수정 후:**
```csharp
public virtual void OnEnd() {
    Context.Timer.Stop();
    //260402 hbk — OPS-01: Action별 택타임 Trace 로그 출력 (D-01: ELogType.Trace + [TAKT] 접두사, D-02: 개별 형식)
    Logging.PrintLog((int)ELogType.Trace, "[TAKT] {0}: {1}ms", Name, Context.Timer.ElapsedMilliseconds);

    Context.State = EContextState.Idle;
}
```

변경 사항:
- `Context.Timer.Stop()` 직후, `Context.State = EContextState.Idle` 직전에 PrintLog 한 줄 추가
- `(int)ELogType.Trace` = 0 을 logID로 사용 (per D-01: 별도 ELogType 추가 안 함)
- 형식: `[TAKT] {Name}: {ElapsedMilliseconds}ms` (per D-02)
- 항상 출력, 조건 분기 없음 (per D-03: ON/OFF 설정 불필요)
- SystemSetting 변경 없음 (per D-03)
- `using FinalVisionProject.Utility;`와 `using FinalVisionProject.Setting;`이 이미 ActionBase.cs 상단에 존재하므로 추가 using 불필요

기존 Logging 인프라(날짜별 파일 분리 + LogDeleteDay 30일 자동 삭제)가 그대로 적용된다 (per D-04).
